# ⌨️ Type Racer

A fast-paced, browser-based typing speed game built with **React** and **Vite**. Challenge yourself to type as fast and accurately as possible — track your WPM, accuracy, and improve over time.

## ✨ Features

- 🏁 **Real-time typing test** — type a displayed passage and get instant feedback
- 📊 **WPM calculation** — words per minute tracked live as you type
- ✅ **Accuracy tracking** — see how precise your keystrokes are
- 🎨 **Visual feedback** — correct and incorrect characters highlighted in real time
- 🔄 **Restart anytime** — reset the test and go again with a fresh passage
- ⚡ **Blazing fast** — powered by Vite for instant hot module replacement during development

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18 or higher
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/shivamkumar1103/type-racer.git

# 2. Navigate into the project directory
cd type-racer

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

The app will be running at **http://localhost:5173** by default.

## 🎮 How to Play

1. Open the app in your browser.
2. Click on the text area or start typing to begin the timer.
3. Type the displayed passage as fast and accurately as you can.
4. Your **WPM** and **accuracy** are calculated when you finish.
5. Click **Restart** to try again!
