# ⌨️ Type Racer Pro

A fast-paced typing speed game built with **React + Vite**. Test and improve your typing speed with real-time WPM tracking, accuracy measurement, and a clean, distraction-free interface.

🔗 **Live Demo:** [type-racer-pro.netlify.app](https://type-racer-pro.netlify.app)

---

## 🚀 Features

- ⚡ Real-time WPM (Words Per Minute) calculation
- 🎯 Accuracy tracking
- ⏱️ Countdown timer
- 🔁 Randomized text passages on each round
- 🎨 Highlighted correct / incorrect keystrokes
- 📱 Responsive design — works on desktop and mobile

## 📦 Getting Started

### Prerequisites

- Node.js `>= 18.x`
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/shivamkumar1103/type-racer.git

# Navigate into the project directory
cd type-racer

# Install dependencies
npm install
```

### Running Locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production

```bash
npm run build
```

The output will be in the `dist/` folder.

### Preview Production Build

```bash
npm run preview
```

## 🌐 Deployment

This project is deployed on **Netlify**.

[![Netlify Status](https://api.netlify.com/api/v1/badges/type-racer-pro/deploy-status)](https://app.netlify.com/projects/type-racer-pro/overview)

To deploy your own fork:

1. Push to GitHub
2. Connect the repo to [Netlify](https://netlify.com)
3. Set build command: `npm run build`
4. Set publish directory: `dist`
